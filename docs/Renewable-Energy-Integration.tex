\section{Project Goal}

The project aims to develop and analyse a simplified AC power flow using IEEE bus electrical network samples. 
The system is being designed as a structured analysis/ engineering web application focused on:

\begin{itemize}
    \item Load flow analysis,
    \item Integration of renewable energy sources,
    \item Optimisation of load flow.
\end{itemize}

\noindent The project is intended both as:

\begin{itemize}
    \item a technical learning platform.
    \item professional development.
\end{itemize}
